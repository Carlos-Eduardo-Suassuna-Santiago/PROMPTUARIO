INFORMAÇÕES GERAIS
------------------
Nome: Promptuario
Versão: 1.0.0
Tipo: Sistema Web de Prontuário Eletrônico
Área: Saúde Pública
Arquitetura: MVT (Model-View-Template)

TECNOLOGIAS UTILIZADAS
----------------------
- Backend: Python 3.11 + Django 4.2
- Frontend: Django Templates + CSS3 + JavaScript
- Banco de Dados: SQLite (dev) / PostgreSQL (prod)
- Testes: Pytest + Coverage
- Infraestrutura: Docker + Docker Compose + Nginx
- CI/CD: GitHub Actions

MÓDULOS IMPLEMENTADOS
---------------------
1. Accounts - Autenticação e gestão de usuários
2. Patients - Cadastro e gestão de pacientes
3. Appointments - Agendamento de consultas
4. Medical Records - Prontuários eletrônicos
5. Reports - Geração de relatórios
